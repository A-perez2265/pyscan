# pyscan
python scanner for Software Eng I
The python scanner is set up to search a static python script, There are 3 different flags risky, deprecated, & validation.
By default if given no flags it will search for all 3, 
to give a flag you can add --flag ( eg. --risky ) 

default with all flags py pyscan.py test.py 

with flag py pyscan.py test.py --risky
